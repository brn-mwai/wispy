# X/Twitter Poster

Post tweets and threads via Twitter API v2. Media attachments supported.

## Badge
Twitter API v2

## Tools
### post_tweet

### post_thread

## Requirements
Twitter API v2 credentials required

### Environment Variables
- `TWITTER_API_KEY`
- `TWITTER_API_SECRET`
- `TWITTER_ACCESS_TOKEN`
- `TWITTER_ACCESS_SECRET`
