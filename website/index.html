How to Set Up

Running in 3 steps

From install to autonomous agent in under 2 minutes

1

Install Globally

Install Wispy via npm, pnpm, or the curl installer

$ npm install -g wispy-ai
  • Works on macOS, Linux, Windows
  • Requires Node.js 20+
  • Auto-installs dependencies
2

Run Setup Wizard

Interactive wizard configures your API keys and channels

$ wispy setup
  • Enter Gemini API key
  • Connect Telegram bot (optional)
  • Configure Vertex AI (optional)
3

Launch Gateway

Start the agent and begin interacting via any channel

$ wispy gateway
  • REST API on localhost:4000
  • Telegram bot starts polling
  • Memory system initialized

What It Does

Marathon Mode

Multi-day autonomous execution. Decomposes goals into milestones, detects infinite loops, self-recovers from errors, and checkpoints progress. Works while you sleep.

Trust Controls

Four levels: auto, notify, approve, deny. You decide what Wispy can do autonomously vs. what needs your approval. Dangerous actions blocked by default.

Thinking Levels

Configure reasoning depth from none (0) to ultra (65K tokens). More thinking = better planning. Marathon uses 24K tokens for milestone decomposition.

Telegram Native

Full bot with inline approval buttons, voice notes, progress updates, and file sharing. Run /marathon and watch it work. Pause, resume, cancel anytime.

Document Factory

Generate LaTeX PDFs with charts (bar, line, pie, scatter), flowcharts, and tables. Professional reports, invoices, proposals‚Äîall from natural language.

Agent Payments

x402 protocol for HTTP-native USDC payments on Base. Wispy can pay for APIs, services, and other agents autonomously. ERC-8004 compliant.

Key Features

How Wispy actually works

Deep dive into the core capabilities powered by Gemini

üèÉ Marathon Mode

Multi-day autonomous execution with intelligent planning, self-verification, and checkpoint-based recovery. Works while you sleep.

  • Smart Planning: Decomposes goals into atomic 15-60 minute milestones using 24K thinking tokens
  • Loop Detection: Hashes actions to detect infinite cycles and forces replanning when stuck
  • Self-Recovery: Up to 3 retry attempts per milestone with error analysis
  • Checkpointing: SHA256 file snapshots enable pause/resume anywhere

üîê Trust Controls

Human-in-loop approval system with 4 trust levels. Critical actions require your explicit approval via Telegram inline buttons.

  • auto: File ops, bash, browser automation ‚Äî approved instantly
  • notify: Messaging, posting ‚Äî logged and notified
  • approve: Payments, emails ‚Äî requires human confirmation
  • deny: Destructive operations ‚Äî blocked entirely

üß† Thinking Levels

Dynamic thinking budget allocation from quick responses to deep 64K-token planning. The agent automatically selects complexity based on task.

  • none: Instant responses for simple queries
  • low (1K): Basic reasoning tasks
  • medium (8K): Standard problem solving
  • high (24K): Complex multi-step tasks
  • ultra (64K): Marathon planning, architecture
